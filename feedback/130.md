### 1. Reformatted Elixir Code

```elixir
defmodule BinaryConverter do
  @doc """
  Converts a list of integers to their binary string representations.

  Negative integers are prefixed with `-` (e.g., `-5` → `"-101"`).
  Zero is converted to `"0"`. Empty lists return an empty list.
  """
  @spec convert_to_binary(list(integer())) :: list(String.t())
  def convert_to_binary(nums) when is_list(nums) do
    Enum.map(nums, &Integer.to_string(&1, 2))
  end
end
```
